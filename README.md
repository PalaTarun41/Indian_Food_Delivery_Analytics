# Indian Food Delivery Analytics (Zomato/Swiggy Style)

A SQL-focused analytics project simulating real-world food delivery platforms like Zomato and Swiggy.

## Dataset Includes
- Orders  
- Customers  
- Restaurants  
- Delivery Partners  

## What You Can Analyze
- Top cuisines
- Delivery time performance
- City-wise order patterns
- Customer segments
- Partner efficiency
- Peak ordering hours

## How to Use
1. Import schema from `sql/database_schema.sql`
2. Load CSV data from `data/`
3. Run all analysis queries in `sql/analysis_queries.sql`

## Tech Stack
- SQL (MySQL/PostgreSQL)
- Python (optional)
- Jupyter Notebook (optional)
